# 🌰 どんぐりリハビリ

リスと一緒にコツコツ続けるリハビリ記録アプリ(PWA)。
仕様書 v0.2 のMVP機能をすべて実装し、three.js製の3Dリスと「毎日起動したくなる工夫」を追加しています。

## 実装ずみの機能

- **タスク管理**:自由入力・カテゴリ・頻度(毎日/曜日指定/週◯回)・メモ・並び替え(ドラッグ)・休止/削除(確認ダイアログつき)
- **デイリーチェック**:今日のタスクをタップで完了 → どんぐり獲得アニメ(ヘッダーへ飛ぶ)+ 実施メモ(回数・角度)
- **目標**:習慣目標/数値目標、進捗バー、現在値の手動更新、達成演出
- **リワード**:どんぐり(1タスク=1個)、コンプリートボーナス(+2)、5段階のリス成長、バッジ8種、レアどんぐり4種
- **ストリーク**:対象タスク全達成で+1、対象なしの日はスキップ扱い、途切れたら0(最長記録は保持)— 仕様書6.4準拠
- **統計・履歴**:月間ヒートマップ(月送り可)、週次達成率グラフ、タスク別達成率、**CSV/JSON書き出し**
- **設定**:リスの名前、文字サイズ3段階、ライト/ダーク/自動、バックアップ書き出し・読み込み・全消去
- **PWA**:manifest+Service Workerでオフライン動作、「ホーム画面に追加」案内つきオンボーディング
- **プライバシー**:全データは端末内のIndexedDBのみ。外部送信なし

## 🐿️ three.jsのリスと「毎日起動したくなる工夫」

- **手作りの3Dリス**(外部モデル不要・オフラインOK):まばたき・呼吸・しっぽの揺れ・首かしげが常時動く
- **なでると喜ぶ**:リスをタップすると跳ねて話しかけてくる
- **現実の時刻で空が変わる**:朝焼け → 昼 → 夕焼け → 星空(夜は星が出ます)
- **現実の季節で景色が変わる**:春=桜色の木/夏=深緑/秋=紅葉/冬=雪景色
- **デイリーログインボーナス**:1日の最初の起動でどんぐり+1&あいさつ
- **タスク完了でジャンプ、全達成でどんぐりの雨**(物理っぽくぽよんと跳ねます)
- **レベルで見た目が進化**:しっぽが立派に → Lv3でマフラー → Lv4でどんぐり貯蔵の山 → Lv5で葉っぱの冠
- **カムバック歓迎**:3日以上あいて再開すると責めずに「おかえり」+専用バッジ

## 動かし方

ES Modulesを使うため、**HTTPサーバー経由**で開いてください(file:// 直開きは不可)。

```bash
# ローカルで試す
cd donguri-rehab
python3 -m http.server 8080
# → http://localhost:8080 を開く
```

### Vercelへのデプロイ(推奨)

1. このフォルダをGitHubリポジトリにpush(またはVercel CLIで `vercel` を実行)
2. Vercelで「New Project」→ リポジトリを選択 → Framework Preset は **Other**(ビルド不要の静的サイト)
3. デプロイされたURLをiPadのSafariで開く

### iPadでの使い方

1. SafariでURLを開く
2. 共有ボタン(⬆︎)→「**ホーム画面に追加**」
3. ホーム画面のどんぐりアイコンから起動(フルスクリーンで動きます)

## ファイル構成

```
donguri-rehab/
├── index.html            画面(5タブ+モーダル)
├── manifest.webmanifest  PWAマニフェスト
├── sw.js                 Service Worker(オフラインキャッシュ+Push受信口)
├── css/style.css         スタイル(森トーン/ダーク/文字サイズ/横向きサイドバー)
├── js/
│   ├── main.js           起動・タブ・ホーム・デイリー演出・設定
│   ├── db.js             IndexedDBラッパー(仕様書5章のデータモデル)
│   ├── game.js           どんぐり・レベル・バッジ・ストリーク(仕様書6章)
│   ├── squirrel.js       three.js製3Dリス(季節・時刻・アニメ・レベル演出)
│   ├── ui.js             トースト・お祝い・どんぐり飛翔・リスのセリフ
│   ├── tasks.js          タスク管理画面
│   ├── goals.js          目標画面
│   ├── collection.js     コレクション画面
│   └── stats.js          統計・履歴画面(書き出し含む)
├── vendor/three.module.js  three.js本体(同梱・オフライン対応)
└── icons/                PWAアイコン(どんぐり顔つき)
```

## 技術メモ

- three.js r160 を同梱(CDN不使用なので機内モードでもリスは動きます)
- リスはプリミティブ(球・カプセル・円錐)を組み合わせた手作りローポリ+Toonマテリアル
- データはすべて IndexedDB。バックアップJSONは仕様書5章のスキーマそのままなので、将来のサーバー同期にも流用できます
